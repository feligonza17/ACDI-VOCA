El objetivo de este documento es empezar a cruzar variables de diferentes bases de datos. En efecto, las bases anteriores (y sus analisis) podian estar presentando problemas de multicolinealidad. Las variables de Gobierno, a pesar de ser diferentes eran parecidas. Por lo tanto, aqui se hara alguna estadistica inferencial mezclando bases de datos para poder ver que puede predecir el indice PAR.

Adicionalmente se busca ver estas mismas estadisticas pero con el resto de variables con las que se ha venido trabajando antes. Es decir: +Reconciliacion +Disculpas +Violencia +Rencor +Memoria (Memoria historica) +Memoria_expectativa (Item creado a partir de la memoria y de la expectativa de esta)
